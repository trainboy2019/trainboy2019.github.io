    Mode hors ligne

    Pour utiliser Miiverse, vous devez préalablement y accéder en étant connecté à Internet et accepter les termes du code de conduite Miiverse.

    Ensuite, même lorsque la console n'est pas connectée à Internet, vous pourrez écrire des messages et les sauvegarder afin de les publier ultérieurement.
