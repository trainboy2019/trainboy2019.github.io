What is freeShop?

It's an open-source homebrew eShop alternative for the Nintendo 3DS. It allows you to browse and install titles you own utilizing title keys.

How to use freeShop

  1. Install the latest version of this software on your preferred CFW.
  2. Put an URL linking to the encTitleKey.bin file on the freeShop's Settings > Update > Auto-update title keys from URL.
  3. You can now use freeShop, enjoy !
